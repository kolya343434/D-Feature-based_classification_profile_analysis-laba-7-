# Лабораторная работа №7. Классификация на основе признаков

Реализован конвейер, в котором каждое изображение символа сводится к вектору признаков (масса, координаты центра тяжести и осевые моменты инерции). Расстояние в этом пространстве нормализуется и переводится в меру близости `1 / (1 + distance)` (нулевое расстояние → единица близости). Последние столбцы отчетов сортируются по убыванию этой меры.

## Архитектура решения
1. **Генерация образца символа.** Шрифт (по умолчанию `arial`/`DejaVu Sans`) используется для отрисовки каждого символа алфавита и строки, чтобы получить одинаковый размер канвы и набор сегментов.
2. **Фичи.** Для каждого символа вычисляются:
   - `mass = sum(pixel) / area`;
   - `centroid_x = ∑ x·pixel / mass` и `centroid_y` аналогично;
   - осевые моменты инерции `∑ (x - centroid)^2·pixel` и `∑ (y - centroid)^2·pixel`, нормализованные на `mass · dimension²`.
3. **Сортировка гипотез.** Для каждого найденного символа рассчитывается евклидово расстояние до всех прототипов и сохраняется список `[(char, closeness), ...]`, отсортированный по убыванию `closeness`.
4. **Отчетность.** Сохраняются:
   - файл `outputs/hypotheses.txt` — реализация формата `1: [("а", 0.997), (…) …]`;
   - `outputs/metrics.md` — табличное сравнение базового и экспериментального прогонов;
   - `outputs/base_render.png` и `outputs/variation_render.png` — визуализация строки.

## Установка
```bash
python -m pip install -r requirements.txt
```

## Запуск
```bash
python src/profile_classifier.py --text "система"
```
Можно задавать шрифт `--font-path`, размер `--font-size`, экспериментальный `--variation-font-size` и собственный алфавит `--alphabet`.

## Что хранится в `outputs/`
- `hypotheses.txt` — набор гипотез (по одной строке на символ, первичны наиболее вероятные буквы).
- `metrics.md` — сравнительный отчет о точности, ошибках и расстоянии Левенштейна для базового и измененного размеров шрифта.
- PNG-изображения строки до и после изменения масштаба.

## Эксперимент
Изменение размера шрифта (от `--font-size` к `--variation-font-size`) проверяет устойчивость меры близости: метрика Левенштейна и количество ошибок записываются в `outputs/metrics.md` в виде таблицы. Для реальных тестов наберите `text` из исторических изображений (см. ниже).

## Источники изображений
Если нужны реальные образцы, используйте коллекцию церковных манускриптов на [slavcorpora.ru](https://www.slavcorpora.ru). Загруженные JPG/PNG можно скормить скрипту, заменив блок `render_text_image` на чтение и сегментацию своих изображений, а затем подсчитать признаки точно так же.
